# 功能列表（未开启web模式）

TIP

功能表会随版本变化，请以自己机器人的帮助页面为准（发送“帮助”获取）

这个功能表是yobot3(无web模式)包含的功能，web模式开启后的功能请在机器人处获取，其他常用功能可以通过安装酷Q插件 (opens new window)来实现。

此页的指令不需要at机器人
如果需要给命令添加前缀，请发送“设置”为机器人设置触发前缀。例如：设置前缀为“#”时，只有发送“#状态”才能触发获取boss状态功能。请注意前缀的全角/半角区别。

(权)标记表示功能需要权限
(自动)标记表示无需触发词，机器人主动推送
(v3.x)标记表示此功能所需的版本号
方括号表示参数可以省略
